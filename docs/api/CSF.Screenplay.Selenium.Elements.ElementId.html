Remarks

This class wraps Selenium's Id(string) functionality. The purpose is provide a type which derives from ITarget, compatible with the Selenium extension's performables. This class also provides opportunity to add a human-readable Name to the element specification. This optional, but recommended, technique improves the readability of Screenplay reports.

In normal circumstances (with valid HTML) this locator should match a maximum of only one element.
